# Contact Management System (Java)

Console-based contact management system developed in Java.

This project demonstrates object-oriented programming principles, data validation and basic data management operations.

Features

- Add new contacts
- Email validation
- Phone number validation
- Prevent duplicate phone numbers
- List all contacts
- Delete contacts

Technologies

Java  
Object Oriented Programming  
ArrayList

Project Structure

Main.java → Program execution and menu  
Contact.java → Contact model  
ContactManager.java → Contact management logic  
Validator.java → Data validation
